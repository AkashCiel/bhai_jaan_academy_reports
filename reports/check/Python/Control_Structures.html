Control structures are used to dictate the flow of execution in a program. They fall into three main categories:

    • Conditional Statements
    • Loops

1. Conditional Statements

Conditional statements allow you to execute specific blocks of code based on whether a condition is true or false. The most common conditional statements in Python are if, elif, and else.

Basic Syntax of Conditional Statements

Hereâ€™s a simple example of how conditional statements work:

```python

age = 18

if age >= 18:

print("You are an adult.")

else:

print("You are a minor.")

``

In this example:

  • The program checks if the variableageis greater than or equal to 18.
  • If the condition is true, it executes the first block of code; if not, it executes the code in theelse` block.

The elif Statement

The elif (short for "else if") statement allows you to check multiple conditions in a sequence. Here's how it works:

```python

grade = 85

if grade >= 90:

print("Grade: A")

elif grade >= 80:

print("Grade: B")

elif grade >= 70:

print("Grade: C")

else:

print("Grade: D")

```

In this example:

  • The program checks the grade variable against multiple thresholds.
  • It executes the corresponding block based on which condition is satisfied first.

2. Loops

Loops enable you to execute a block of code repeatedly as long as a condition is true or for a specified number of iterations. The two most common types of loops in Python are for loops and while loops.

For Loops

A for loop is used to iterate over a sequence (like a list, tuple, string, or dictionary). Hereâ€™s an example:

```python

fruits = ["apple", "banana", "cherry"]

for fruit in fruits:

print(fruit)

```

In this example:

  • The loop iterates through each item in the fruits list and prints it.

Using range()

You can also use for loops with the range() function to execute a block of code a specified number of times:

python

for i in range(5):

print(i)

This will print numbers from 0 to 4.

While Loops

A while loop continues to execute as long as its condition remains true. Hereâ€™s an example:

```python

print(count)

count += 1

```

In this example:

  • The loop runs as long as count is less than 5, incrementing count by 1 on each iteration.

3. Control Flow Statements

These include statements that change the flow of execution in your program. Key examples include break, continue, and pass.

Break Statement

The break statement is used to exit a loop prematurely. For example:

python

for i in range(10):

if i == 5:

break

print(i)

This will print numbers 0 through 4 and then exit the loop when i becomes 5.

Continue Statement

The continue statement skips the current iteration and moves to the next one:

python

for i in range(5):

if i == 2:

continue

print(i)

This will print numbers 0, 1, 3, and 4, skipping the number 2.

Pass Statement

The pass statement is a placeholder that does nothing. It is often used when a statement is syntactically required but you do not want to execute any code.

python

for i in range(5):

if i == 2:

pass # Do nothing for i = 2

print(i)

This will print all numbers from 0 to 4, treating the case for i = 2 without any action.
