    • Conditional Statements: These allow the program to execute certain blocks of code based on specific conditions.
    • if statements
    • if-else statements

    • for loops

    Control Flow Statements: These modify the execution flow of the loops.

1. Conditional Statements

Conditional statements allow the programmer to execute specific blocks of code based on whether a condition is true or false. They are fundamental for decision-making in programs.

1.1. if Statements

The simplest form of a conditional statement is the if statement. It evaluates a condition and executes the block of code within it if the condition is true.

Example:

python

age = 18

if age >= 18:

print("You are eligible to vote.")

1.2. if-else Statements

An if-else statement provides a choice between two paths: one path if the condition is true and another if it is false.

Example:

python

age = 16

if age >= 18:

print("You are eligible to vote.")

else:

print("You are not eligible to vote yet.")

1.3. if-elif-else Statements

For multiple conditions, the if-elif-else statement allows you to check several different conditions sequentially.

Example:

python

marks = 85

if marks >= 90:

print("Grade: A")

elif marks >= 80:

print("Grade: B")

elif marks >= 70:

print("Grade: C")

else:

print("Grade: D")

1.4. Real-World Application of Conditional Statements

Conditional statements are widely used in applications where decisions must be made based on user inputs or other conditions. For example, in a voting application, you might check the age of a user to determine their eligibility to vote. Similarly, in an online shopping platform, you could use conditional statements to apply discounts based on the user's membership status.

2. Loops

Loops allow you to execute a block of code repeatedly based on a condition. They are essential for tasks that require iteration.

2.1. for Loops

A for loop iterates over a sequence (like a list or string) and executes a block of code for each item in that sequence.

Example:

python

for i in range(5):

print(f"Iteration {i}")

2.2. while Loops

A while loop continues to execute as long as a specified condition is true.

Example:

python

count = 0

while count < 5:

print(f"Count is {count}")

count += 1

2.3. Real-World Application of Loops

Loops are invaluable in scenarios where you need to process collections of data. For instance, in data analysis, you might iterate over a dataset to perform calculations or transformations on each entry. In web scraping, loops can be used to navigate through multiple pages of a website.

3. Control Flow Statements

Control flow statements alter the behavior of loops and conditional statements, providing more control over program execution.

3.1. break Statement

The break statement is used to exit a loop prematurely when a certain condition is met.

Example:

python

for i in range(10):

if i == 5:

break

print(i)

3.2. continue Statement

The continue statement skips the current iteration and moves to the next iteration of the loop.

Example:

python

for i in range(5):

if i == 2:

continue

print(i)

3.3. pass Statement

The pass statement is a null operation; it is a placeholder that allows for syntactically required code where nothing needs to be executed.

Example:

python

for i in range(3):

if i == 1:

pass # Placeholder for future code

print(i)

3.4. Real-World Application of Control Flow Statements

Control flow statements are particularly useful in scenarios where you need to manage the flow of loops effectively. For example, in a game, you might use the break statement to exit a loop when the player wins. The continue statement could skip certain actions under specific game conditions.
