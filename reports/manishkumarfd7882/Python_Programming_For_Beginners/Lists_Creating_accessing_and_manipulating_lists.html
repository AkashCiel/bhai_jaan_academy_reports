Key Concepts:

What is a List?

A list in Python is an ordered collection of items that can be of different types, including numbers, strings, or even other lists. Lists are defined by enclosing their elements in square brackets [] and separating them with commas.

Example:

python

fruits = ["apple", "banana", "cherry"]

Key Characteristics of Lists:

    • Ordered: The items in a list maintain their order. The first item has an index of 0, the second item has an index of 1, and so on.
    • Mutable: You can change the items in a list after it has been created. This means you can add, remove, or modify items.
    • Heterogeneous: A list can contain multiple data types within the same collection.

Creating Lists

Lists can be created in several ways:

    • Using Square Brackets: As shown above.
    • Using the list() Constructor: You can create a list from any iterable (like strings or tuples).

Example:

python

numbers = list((1, 2, 3, 4, 5)) # Creates a list from a tuple

    • Empty List: You can also start with an empty list and add items later.

Example:

python

empty_list = []

Accessing List Elements

You can access elements in a list using indexing:

Example:

python

print(fruits[0]) # Outputs: apple

print(fruits[1]) # Outputs: banana

Negative Indexing

Python also allows negative indexing, where -1 refers to the last item, -2 to the second last, and so on.

Example:

python

print(fruits[-1]) # Outputs: cherry

Slicing Lists

You can retrieve a portion of a list using slicing syntax, which is defined by list[start:end].

Example:

python

print(fruits[0:2]) # Outputs: ['apple', 'banana']

Manipulating Lists

Now that we have a foundational understanding of how to create and access lists, letâ€™s explore various manipulation techniques:

Adding Elements to a List

    Using append(): Adds an item to the end of the list.

    python

    fruits.append("orange")

    Using insert(): Adds an item at a specified index.

    python

    fruits.insert(1, "kiwi") # Adds kiwi at index 1

    Using extend(): Adds multiple items to the end of the list.

    python

    fruits.extend(["grape", "melon"])

Removing Elements from a List

    Using remove(): Removes the first occurrence of a specified value.

    python

    fruits.remove("banana")

    Using pop(): Removes an item at a specified index (or the last item if no index is specified), and returns it.

    python

    last_fruit = fruits.pop() # Removes and returns the last item

    Using clear(): Removes all items from the list.

    python

    fruits.clear()

Modifying Elements in a List

You can change the value of a specific item by referencing its index:

Example:

python

fruits[0] = "blueberry" # Changes the first item to blueberry

Looping Through Lists

You can use loops to iterate through the items in a list, accessing each element:

Example:

python

for fruit in fruits:

print(fruit)
